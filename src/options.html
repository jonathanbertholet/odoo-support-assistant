Gemini

Create a key in Google AI Studio. The key is stored in chrome.storage.sync on this profile. The extension sends your extracted ticket text to Google’s generative language API. See privacy.md in the repository.
Override only if your project uses a different model id.

Custom Odoo site

Odoo Online (*.odoo.com, *.odoo.sh) and localhost are already included. If your team uses a private URL, add the origin and approve the permission prompt so the AI brief button can be injected on that site.

Only add hostnames you trust. The same-origin Odoo data you view is processed locally and, when you run Summarize, sent to your configured Gemini model.

Examples: https://odoo.mycompany.com, http://10.0.0.4:8069 (include scheme and, if not 443/80, the port)
