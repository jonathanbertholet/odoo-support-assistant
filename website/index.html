Odoo Support Assistant

Chrome extension (Manifest V3) — AI briefs next to Odoo chatter.

What it does

Reads the current record’s description, chatter, and optional form context, then produces a structured support triage brief (summary, questions, evidence, timeline, and more). Your Gemini API key stays in the extension — no separate backend required.

View on GitHub

Highlights

  • Inline launcher when a chatter with Send message is present
  • Summary, timeline, settings, copy markdown, estimated time saved
  • Star tasks to the toolbar popup; star PRs in the brief
  • Local cache of briefs with regenerate when content changes
  • Built-in Odoo.com / Odoo.sh / common localhost dev ports; add custom origins in options

Install (dev)

Clone the repo, open chrome://extensions, enable Developer mode, Load unpacked, and select the folder that contains manifest.json. Add your Gemini API key in extension options.
