Developers

Install unpacked in a few minutes

No build step required for day-to-day use—load the extension folder and add your API key.

  1. 1

    Clone the repository locally.

  2. 2

    Open chrome://extensions, enable Developer mode, then choose Load unpacked.

  3. 3

    Select the directory that contains manifest.json (the src folder in this project).

  4. 4

    Open extension options, paste your Gemini API key, and save. Reload private Odoo tabs after granting site access.
